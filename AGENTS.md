# AI Agent Context

This file contains context and instructions for AI agents working on this repository.

## Project Overview
This project holds configuration files for Marlin firmware for driving 3D printers. Each folder is named based on the name of the printer and contains `Configuration.h` and `Configuration_adv.h` (C++ configuration files used by Marlin), along with an image file showing the pinout of the control board.

## Repository Structure
- `Anet AM8 black/`: Configuration files for the Black Anet AM8 printer.
- `Anet AM8 silver/`: Configuration files for the Silver Anet AM8 printer.
- `assets/`: Contains utility files:
    - `FAT 232 driver.rar`: Driver required to flash firmware via USB.
    - `U8glib.zip`: Library required for Marlin.

## Conventions
- This repository stores Marlin firmware configurations.
